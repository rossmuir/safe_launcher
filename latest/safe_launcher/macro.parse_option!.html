macro_rules! parse_option {
    ($output:expr, $err_statement:expr) => {
        $output.ok_or($crate::errors::LauncherError::SpecificParseError($err_statement.to_string()))
    }
}

This macro is intended to be used in all cases where we get a None out of Option and want to package it into safe_launcher::errors::LauncherError::SpecificParseError(String). This is useful because there may be miscellaneous erros while parsing through a valid JSON due to JSON not conforming to certain mandatory requirements. This can then be communicated back to the JSON sending client.

Examples

fn g() -> Result<(), safe_launcher::errors::LauncherError> {
    let mut remaining_tokens = vec![];
    let _module = try!(parse_option!(remaining_tokens.pop(), "Invalid endpoint - Module token not found."));

    Ok(())
}

fn main() {
    if let Err(err) = g() {
        println!("{:?}", err);
    }
}
