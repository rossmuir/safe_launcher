Since the entire core library is purely event driven, this module provides types and primitives that an interfacing crate (executable or another library) should make use of to easily tap into the information and events of interest happening in the core.

Observer

All codes interfacing with Launcher are expected to be coded in observer pattern style for collecting notifications from core that they are interested in. This is a type alias for all events that Launcher will fire to interested listeners. Interfacing codes are expected to construct these senders and register them with appropriate core modules. The registration facility will be available with the main launcher module.
